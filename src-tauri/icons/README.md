# 图标文件说明

请在此目录放置以下图标文件：

- `32x32.png` - 32x32 像素 PNG 图标
- `128x128.png` - 128x128 像素 PNG 图标  
- `128x128@2x.png` - 256x256 像素 PNG 图标（2倍分辨率）
- `icon.icns` - macOS 图标文件
- `icon.ico` - Windows 图标文件

可以使用在线工具（如 https://icon.kitchen/ ）生成这些图标。

## 建议的图标设计

BooQ 的图标可以采用：
- 📚 书本 + ❓ 问号的组合
- 主色调：蓝色 (#0d6efd) 或紫色
- 简洁扁平化风格
